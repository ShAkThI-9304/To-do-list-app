# 📝 Shakthi's To-Do List App

A stylish, responsive, and feature-rich **To-Do List Web App** built with **HTML**, **CSS**, and **JavaScript**.
The app allows you to **add, edit, delete, and mark tasks as completed**, while tracking your progress visually.
Your data is **saved in the browser using Local Storage**, so tasks stay even after a page refresh.

## ✨ Features

* **Add New Tasks** with a simple input field.
* **Edit Tasks** before marking them as completed.
* **Delete Tasks** instantly.
* **Track Progress** with a visual progress bar and completed/total count.
* **Celebration Animation** 🎉 when all tasks are completed.
* **Local Storage Support** — your tasks stay after refresh.
* **Responsive Design** for desktop and mobile.

---

## 🛠 Tech Stack

* **HTML5** – App structure
* **CSS3** – Stylish UI with glassmorphism and responsive design
* **JavaScript (ES6)** – Core functionality and interactions
* **Font Awesome** – Icons
* **Confetti.js** – Celebration effect
